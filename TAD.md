# Types abstraits de données (TAD) utilisés

## Files

### Système de crafting (fabrication)
_InventorySystem.py_

On utilise une file pour encoder le système de crafting à la chaîne; il faut finir de fabriquer un item pour pouvoir commencer à fabriquer le suivant.

### Système de rotation d'armes
_Objects.py_

On utilise une file pour la rotation des armes du personnage, pour maintenir toujours le même ordre entre les différentes armes.
